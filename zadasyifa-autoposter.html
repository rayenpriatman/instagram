👥 Kelola Akun Instagram (Multi-Akun)

Tambahkan beberapa akun Instagram. Pilih salah satu sebagai "Akun Aktif" — akun itu yang dipakai saat posting dari tab Feed/Reels. Field "Meta Graph API" di bawah ini otomatis terisi dari akun aktif yang dipilih.

🔐 Meta Graph API (akun aktif saat ini)

Meta Graph API Explorer

17 digit dari Meta Business Suite

Dari Graph API Explorer → me?fields=id

Token khusus Threads dari Threads API → Token Generator (diawali THAA...)

URL Cloudflare Worker proxy untuk bypass CORS. Deploy dari file threads-proxy-worker.js

🐦 X (Twitter) API

URL Cloudflare Worker proxy untuk bypass CORS. Sama seperti Threads proxy.

Dari developer.twitter.com → Keys & Tokens → OAuth 1.0 Keys. Pastikan permission Read and Write.

🤖 Anthropic API Key (Caption AI)

Dari console.anthropic.com → API Keys → Create Key. Opsional — tanpa ini caption pakai template otomatis.

🖼️ Upload Foto (ImgBB — gratis)

Daftar gratis di api.imgbb.com → Get API key

🗂️ Google Drive (Ambil Konten Otomatis)

Ambil dari URL folder: drive.google.com/drive/folders/FOLDER_ID_DI_SINI. Folder harus di-share "Siapa saja yang memiliki link → Melihat".

Buat di Google Cloud Console → Credentials → Create API Key, lalu aktifkan "Google Drive API" di library.

🎲 Template Caption Acak

Isi tiap bagian, 1 baris = 1 pilihan. Tiap kali generate, sistem pilih 1 baris acak dari Judul, 1-2 baris acak dari Isi, 1 baris acak dari Penutup, lalu gabung hashtag acak — jadi captionnya beda-beda tiap posting. Ini dipakai otomatis kalau file Google Drive tidak punya .txt caption sendiri, dan bisa juga dipanggil manual dari tab Feed.

🎬 Upload Video Reels (Cloudinary)
✅ Cloud name Anda: di5bhfcl2 — sudah terisi otomatis!
Hanya perlu buat Upload Preset sekali di Cloudinary.

Terlihat di kiri atas dashboard Cloudinary

Settings → Upload → Upload Presets → Add preset → Unsigned

Cara buat Upload Preset:
1. Buka console.cloudinary.com/settings/upload
2. Scroll ke bawah → Upload presets → klik Add upload preset
3. Signing mode → pilih Unsigned
4. Klik Save → catat nama preset-nya
📖 Panduan Setup
  1. Isi Access Token dari Meta Graph API Explorer
  2. Isi Instagram Account ID (17 digit) dari Meta Business Suite
  3. Daftar ImgBB → isi API key untuk posting foto
  4. Isi Cloud Name, API Key & Secret Cloudinary untuk upload video Reels
  5. Klik Simpan → Test koneksi → siap!
